# vue-final-offcanvas

A better Offcanvas for Vue.

No special dependencies, no jquery, no bootstrap, just VueJS and CSS goodness.

## Requirements

- [Vue.js](https://github.com/vuejs/vue)

## Installation

```shell
$ npm install vue-final-offcanvas
// OR
$ yarn add vue-final-offcanvas
```
